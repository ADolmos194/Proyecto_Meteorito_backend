Django==4.0.3
# - Framework web para Python, utilizado para el desarrollo de aplicaciones web completas.
# - En este caso, se ha elegido una versión estable de Django 4.x.

djangorestframework
# - Framework para construir APIs RESTful en Django.
# - Proporciona las herramientas necesarias para crear y administrar endpoints en el backend.

django-cors-headers
# - Middleware para habilitar CORS en Django.
# - Permite que el backend Django acepte solicitudes de frontend desde diferentes orígenes.

djangorestframework-simplejwt
# - Paquete para la autenticación mediante JWT en Django REST Framework.
# - Facilita la implementación de autenticación y autorización mediante tokens.

psycopg2-binary
# - Conector para bases de datos PostgreSQL en Python.
# - Se usa para interactuar con bases de datos PostgreSQL desde Django.

pandas
# - Librería para manipulación de datos.
# - Muy utilizada para análisis de datos, procesamiento de grandes volúmenes de datos y operaciones sobre DataFrames.

openpyxl
# - Herramienta para leer y escribir archivos Excel (.xlsx).
# - Ideal para la creación o modificación de archivos Excel dentro de tu aplicación.

xlsxwriter
# - Librería para generar archivos Excel (.xlsx).
# - Más orientada a la creación de informes complejos y archivos de salida Excel.

requests
# - Librería para hacer solicitudes HTTP de manera sencilla.
# - Usada para consumir APIs, realizar peticiones GET, POST, PUT, DELETE, etc.

Jinja2
# - Motor de plantillas para Python.
# - Usado principalmente por Django para renderizar templates HTML y otros documentos.

xhtml2pdf
# - Convierte HTML a PDF.
# - Utilizada para generar documentos PDF a partir de contenido HTML.

Office365-REST-Python-Client
# - Cliente REST para interactuar con los servicios de Office 365.
# - Usado para integrarse con servicios como SharePoint, OneDrive, Outlook, etc.

reportlab
# - Librería para generar documentos PDF en Python.
# - Usada para crear informes, facturas o cualquier otro documento en formato PDF.

weasyprint
# - Conversión de HTML y CSS en PDF.
# - Permite generar archivos PDF de alta calidad a partir de contenido HTML.

num2words
# - Convierte números a palabras (por ejemplo, 123 a "ciento veintitrés").
# - Utilizada para representar valores numéricos como texto, como en informes o facturas.

mysqlclient
# - Conector para bases de datos MySQL en Python.
# - Utilizado para conectar y realizar operaciones en bases de datos MySQL desde Django.
